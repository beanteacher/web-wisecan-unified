# PRD — WiseCan 통합 메시징 서비스

> 작성일 2026-06-01

---

## 한 줄 비전

**AI 시대에 가장 저렴하고 가장 통합 친화적인 한국 메시지 발송 서비스** — 하나의 계정·하나의 키·하나의 잔액으로 SMS / LMS / MMS / 카카오 알림톡 / RCS를 보낸다.

## 왜 만드는가 (시장 문제 3가지)

1. **AI Agent 통합 시 토큰 비용 폭증.** 기존 메시지 SaaS는 MCP만 제공하거나 CLI가 아예 없어, AI Agent가 발송·이력 조회 한 번에 4,000~6,000 토큰을 태운다. → **MCP + CLI를 1:1 동등 제공.** 자동화 루프는 CLI(`wsc ...`)로 위임해 같은 작업을 600~900 토큰으로 끝낸다(5~10배 절감).
2. **중계사·라우팅이 회원에게 새어 나간다.** LG CNS·KT·인포뱅크 등 카카오/RCS 중계사 식별자가 응답·이력에 노출돼 회원 혼란과 운영 협상력 약화를 동시에 일으킨다. → **중계사·라우팅·내부 채널 ID는 응답·UI·로그·이력 어디에도 노출하지 않는다.** 회원 인지 단위는 채널(SMS / LMS / MMS / 카카오 알림톡 / RCS)까지로 한정한다.
3. **잔액 소진 시 발송 일괄 차단으로 운영 클레임이 폭증한다.** → **사전 평가 + 분기 처리.** 자동결제 → 후불 → 부분 발송/취소 순으로 라우팅하고, 이미 적재된 발송은 완주시킨다.

## 누구를 위해 (페르소나 3종)

| 페르소나 | 핵심 니즈 | 본 서비스가 주는 가치 |
|---|---|---|
| **개인 개발자 / 1인 사업자** (`MEMBER`) | 본인 인증만으로 즉시 발송, 코드 통합 | 본인 인증 → 발신번호 등록 → API Key 발급 → SDK·CLI로 즉시 발송. 셀프서비스 완결. |
| **회사 마스터** (`COMPANY_MASTER`) | 사업자 등록증으로 회사 단위 발신번호·결제 통합, 직원 권한 분리 | 회사 마스터 + 하위 계정 모델, 키별 스코프(12종)로 직원별 데이터·발송 권한 통제. |
| **AI Agent를 운영하는 개발자** | 대화형으로 발송·이력 조회 자동화하되 토큰 비용은 최소화 | MCP(자연어 단발) + CLI(자동화 루프) 동등 제공. 보안 민감 액션(키 발급·발신번호 등록·결제)은 의도적으로 MCP 비노출 — 자연어 추론으로 인한 우발 호출 차단. |

> 부가: 가입 결정 전 회원과 동일 UI를 더미 데이터로 만져볼 수 있는 **비회원 체험 모드**로 전환율을 끌어올린다.

## 핵심 가치 (2가지)

- **AI 토큰 비용 5~10배 절감** — MCP/CLI 동등성 + CLI 위임 권장으로 회원 본인 AI Agent 비용을 구조적으로 낮춘다(본 서비스는 측정·과금하지 않음).
- **운영 복잡도 추상화** — 중계사·라우팅·KISA 사전 등록·잔액 부족 분기를 회원이 의식하지 않게 감춘다. 회원은 채널과 메시지만 다룬다.

## 성공 지표 (KPI 3개)

| # | 지표 | 목표 (출시 후 3개월) | 측정 출처 |
|---|---|---|---|
| **K1. 시장 진입** | MAU (월 1회 이상 발송한 회원) | **≥ 200명** + 누적 캐시 충전 **≥ 5,000만 원** | 발송·결제 로그 |
| **K2. 차별화 검증** | AI Agent 발송·조회 중 CLI 위임 비율 | **≥ 50%** | 진입점 메타 로그 |
| **K3. 운영 신뢰성** | 발송 테이블 적재 성공률 / 발송 API 가용성 | **≥ 99.9% / ≥ 99.5%** | Prometheus (메트릭·알림 정의는 `06_OBSERVABILITY.md §2·§3`) |

> 보조 지표(체험→가입 전환율 ≥ 15%, 자동충전 결제 실패율 ≤ 1%, 1:1 문의 24h 처리)는 `04_PROJECT_PLAN.md §리스크·운영 게이트`로 위임.

## MVP 범위 한 줄

회원·발신번호·키·발송·결제(선불 + 자동충전 + 후불) + SDK(Python)·MCP·CLI(`wsc`) + 운영자 콘솔 + 체험 모드. **구독·세금계산서·특수부가·사설 연동·글로벌 발송은 1차 범위 외.** 세부는 `02_FEATURE_SPEC.md`.
