구현 목표

JWT 인증 기반 웹 콘솔 발송 엔드포인트를 신규 구현한다. API Key 기반 /dispatch/send와 완전 분리하여 /console/send 경로로 제공하며, 5채널(SMS·LMS·MMS·KAKAO·RCS) 발송·일괄 발송(최대 100,000행)· 예약 발송·예약 취소·예약 목록 조회를 포함한다.

POST /console/send —단건 발송 (5채널, 수신자 최대 1,000개)
POST /console/send/bulk —일괄 발송 (CSV 파싱 결과 최대 100,000행, groupId 부여)
POST /console/send/scheduled —예약 발송 등록 (scheduledAt = requestedAt)
GET /console/send/scheduled —PENDING 상태 미래 예약 목록 조회 (페이지네이션)
DELETE /console/send/scheduled/{sendId} —예약 취소 (PENDING + 발송 시각 이전만)
