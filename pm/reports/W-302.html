결론 및 잔여 이슈

W-302 CLI wsc 구현이 완료되었습니다. 발송 5채널 단건·일괄·예약, 이력 조회, API Key 관리, 발신번호 관리, 설정, docs·snippet 등 명령 14그룹 26 하위 명령을 Node.js 내장 모듈만으로 구현하였으며, 단위 테스트 38종이 모두 통과하였습니다. 로컬 SSL 인터셉션 환경에서 외부 패키지 설치가 불가능하므로 commander 의존성을 제거하고 경량 arg 파서를 자체 구현하였습니다. pkg 기반 3 OS(win/mac/linux) 빌드 구성과 자동 업데이트 메커니즘도 갖추었습니다.

잔여 이슈 / CI 위임 항목
  • pkg 바이너리 실제 빌드는 로컬 SSL 인터셉션으로 npm install 불가 — CI 환경에서 수행 필요 (dist/ 산출물 없음)
  • wsc auth login API Key 검증 — 현재 저장만 하고 실제 ping 검증 없음. W-304 발송 이력 조회 연동 후 보완 권장
  • OS 키체인 연동 — 현재 chmod 600 파일 방식. keytar 등 키체인 라이브러리는 SSL 해소 후 추가 가능
  • wsc send bulk CSV 파일 입력 지원 — 현재 --destaddrs 쉼표 방식만 지원, --csv-file 옵션은 차기 스프린트 대상
