  • 결제 / 환불 / 충전 — PaymentTool, RefundTool, ChargeTool 미생성

각 도구의 @Tool(description = "...") 에 한계 안내 문구를 포함하여, LLM이 불가능한 작업을 요청할 경우 사용자에게 웹 콘솔 또는 CLI를 안내하도록 설계되었습니다.

결론 및 잔여 이슈

W-303 MCP 서버 도구 13종 구현이 완료되었습니다. backend/src/main/java/com/wisecan/unified/mcp/ 에 10개 도구 클래스(GuideTool·SnippetTool·SendTool·HistoryTool·CallbackTool·ApiKeyTool·BalanceTool·UsageTool·TemplateTool·BrandTool), service/dispatch/SendHistoryService.java 신규 서비스, config/McpConfig.java 갱신, 테스트 2종이 추가되었습니다 (총 13개 신규 파일 + 1개 수정). 비노출 정책(키 발급/폐기·발신번호 등록/삭제·결제·routingMeta)은 도구 미생성 + DTO 필드 제외 이중으로 보장합니다. SDK/CLI/MCP 동등성은 모든 실행 도구가 동일 서비스 메서드를 직접 호출하는 구조로 달성되었습니다.
