완료 기준 (DoD) 달성 현황
  • §12.3 강제 정지 — Member.suspend() + 발신번호 일괄 해지 + API Key 일괄 REVOKED + 감사 로그 단일 트랜잭션
  • §12.3 강제 해지 — Member.terminate() + 동일 연쇄 처리, 해지 후 재활성화 불가 보장
  • §12.3 정지 해제 — SUSPENDED → ACTIVE 전이, 발신번호·키 복원 없음 (정책 준수)
  • §12.3 운영자 보호 — ADMIN/SUPER_ADMIN 역할 회원 통제 금지 (assertNotAdmin)
  • §12.3 Append-only 감사 로그 — MemberControlAuditLog 전 컬럼 updatable=false
  • §12.4 라우팅 매핑 — (memberId, channel) 유니크 제약 + Upsert 패턴 + 회원 완전 비노출
  • §12.5 환불 목록 조회 — 전 회원 환불 건 + 이메일 조인
  • §12.5 캐시 강제 적립 — ADMIN_GRANT ChargeBalance INSERT + ADMIN_CREDIT 원장 + 캐시 무효화 이벤트
  • §12.5 캐시 강제 차감 — FIFO(만료 임박 우선) 잔액 소진 + 잔액 부족 예외 + ADMIN_DEBIT 원장
  • §12.5 정산 보고서 — 기간별 회원별 충전·환불·사용·잔액 집계
  • 단위 테스트 21개 — @ExtendWith(MockitoExtension) + @InjectMocks + 서비스 3개 전체 커버
  • 공유 파일 수정 최소화 — SecurityConfig 미수정, MemberStatus·billing enum 열거값 추가만 수행
