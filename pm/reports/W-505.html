결론 및 잔여 이슈

W-505 CS 도메인을 신규 cs/support 패키지로 완전 격리하여 구현 완료. 1:1 문의(생성·답변·상태), 챗봇(FAQ 기반 Stub), FAQ 관리, 공지사항 4개 기능과 24h SLA 측정(answeredAt 기록 + 통계 API)이 DoD를 충족함. 단위 테스트 19개 작성(도메인·서비스·챗봇 계층 TDD 순서 준수). FE는 FSD 구조로 features/cs 슬라이스를 신설하고 CS 통합 페이지(4탭) + ChatbotWidget 플로팅 위젯을 완성.

잔여 이슈 / 후속 과제
  • 로컬 SSL 인터셉션으로 gradle 빌드 불가 — CI 파이프라인에서 통합 테스트 수행 필요
  • 챗봇 키워드 매칭은 단순 LIKE 기반 Stub — LLM 연동 시 ChatbotService 교체 또는 확장 예정
  • InquiryRepository SLA 쿼리의 TIMESTAMPDIFF는 MySQL 전용 — H2 Oracle 모드 테스트 시 네이티브 쿼리 분기 필요
  • 관리자 콘솔 UI(문의 처리 화면, FAQ/공지 CRUD)는 W-503 운영자 어드민 범위에서 추가 구현 권장
  • 파일 첨부 기능(문의 이미지 업로드)은 MVP 미포함 — 추후 스프린트 과제
