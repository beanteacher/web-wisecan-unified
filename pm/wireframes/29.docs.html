소개

WiseCan은 SMS, LMS, MMS, 카카오 알림톡, RCS를 단일 API로 통합한 메시지 발송 서비스입니다. Python SDK와 셸 CLI를 통해 어떤 환경에서도 쉽게 연동할 수 있으며, AI 에이전트와의 통합도 지원합니다.

지원 채널

  • SMS — 단문 (90바이트 이하)
  • LMS — 장문 (2,000바이트 이하, 제목 포함)
  • MMS — 이미지 첨부 (300KB 이하, 720×1280px)
  • 카카오 알림톡 — 승인된 템플릿 기반 발송
  • RCS — 브랜드 등록 후 사용 가능

빠른 시작

아래 예시는 SMS를 발송하는 가장 간단한 코드입니다. API Key는 콘솔의 API Key 메뉴에서 발급할 수 있습니다.

send_sms.py 
import wisecan

client = wisecan.Client(api_key="wc_live_xxxxxxxxxxxx")

result = client.sms.send(
    from_number="02-1234-5678",
    to="010-9876-5432",
    message="안녕하세요! WiseCan 발송 테스트입니다."
)

print(result.message_id)  # snd_abc123def456
terminal 
# wsc CLI 설치
$ pip install wisecan-cli

# API Key 설정
$ wsc config set api-key wc_live_xxxxxxxxxxxx

# SMS 발송
$ wsc sms send \
    --from "02-1234-5678" \
    --to "010-9876-5432" \
    --message "안녕하세요! WiseCan 발송 테스트입니다."

✓ 발송 완료 — snd_abc123def456
REST API — POST /messages/sms 
// Request
{
  "from": "02-1234-5678",
  "to": "010-9876-5432",

// Response 200 OK
{

  "channel": "SMS",
  "created_at": "2026-06-01T10:30:00Z"
}

인증

모든 API 요청에는 Authorization: Bearer {API_KEY} 헤더가 필요합니다. API Key는 콘솔에서 테스트 키와 운영 키를 각각 발급할 수 있으며, 스코프(권한 범위)를 키별로 설정할 수 있습니다.

API Key 환경 변수 설정

코드에 직접 하드코딩하지 말고 환경 변수로 관리하세요.
