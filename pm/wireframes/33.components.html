Alert 모달

라이트 surface · 일반 글자 weight · 좌측 아이콘 + 우측 텍스트 · 푸터 우측 액션 · 아이콘은 inline SVG (Lucide 스타일) · 자동 사라짐 X — [확인] 강제.

1. 정보 알림 — 확인 1개

처리가 완료되었습니다

설명 한두 줄. 다음 행동이 명확하면 안내 문장 1개 포함.

사용 예 — 가입 완료 · 저장 완료 · 발송 접수 · 비밀번호 변경

2. 확인 분기 — 취소 + 확인

계속하시겠어요?

잠재 위험 / 비가역 결과를 한 문장으로 명시 후 회원 명시 확인 받음.

사용 예 — 동일 명의 안내 · 발송 시 광고 표기 동의 · 스코프 변경 확인

3. Destructive — 취소 + 삭제

정말 삭제하시겠어요?

되돌릴 수 없습니다. 영향 범위(연결된 리소스 N개)를 함께 명시.

사용 예 — 계정 탈퇴 · API Key 폐기 · 발신번호 삭제 · 하위 계정 삭제

4. Commerce — 취소 + 결제

결제를 진행할까요?

금액 · 결제수단 · 처리 후 상태(잔액 변화 등)를 명시 후 확인 받음.

사용 예 — 캐시 충전 · 자동충전 활성 · 후불 청구서 결제

사용 규칙

  • 자동 사라짐 없음 — 회원이 [확인]을 누를 때까지 유지. 메시지 누락 방지.
  • 아이콘 4종 — Info(블루 check) · Warning(앰버 triangle) · Destructive(레드 trash) · Commerce(오렌지 card). 그 외 변형 금지.
  • 제목은 한 문장 결론 — "처리가 완료되었습니다" · "정말 삭제하시겠어요?". "처리 중 오류 발생" 같은 모호한 제목 금지.
  • 버튼은 1~2개. 우측 정렬. 주 액션이 항상 마지막(오른쪽).
  • 필드 단위 검증은 인라인 — 이메일 형식 같은 입력 에러는 필드 옆 빨간 텍스트. 모달 남발 금지.
